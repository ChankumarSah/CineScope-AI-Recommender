CineScope — AI Movie Recommender

An elegant, content-based movie recommendation system powered by machine learning and Streamlit.

⭐ Overview

CineScope is an AI-powered movie recommendation system that suggests similar movies using content-based similarity.
Instead of relying on user ratings, it analyzes descriptions, genres, directors, and language to understand what makes movies alike.

The app provides a modern, cinematic UI, fast recommendations, and real-time poster previews using the OMDb API.
This project demonstrates strong ML, NLP, UI development, and deployment-ready application design.

🚀 Features

🎬 Content-Based ML Model using TF-IDF embeddings + cosine similarity

⚡ Fast Movie Recommendations with k-Nearest Neighbors

🖼️ Automatic Movie Poster Fetching via OMDb API

🧠 Smart Vector-based Similarity Search

🌐 Interactive Streamlit UI with classy design

🔧 Configurable file paths, API keys, and advanced settings

📦 Clean architecture & easy-to-extend ML pipeline
